# Clicker Heroes Save Editor (CHSE)

**Clicker Heroes** 用のモダンでスタイリッシュなセーブデータエディタです。
白と黒を基調とした「モノクローム・スタイリッシュ」なデザインで、直感的な操作感を提供します。

🔗 **[CHSE を開く (GitHub Pages)](https://nexy451z.github.io/CHSE/)**
🔗 **[もしくは（オフライン動作ソフト）](https://github.com/Nexy451z/CHSE/releases/download/v1.0.0/CHSE-win32-x64.zip)**

##  機能 (Features)
- **資源編集 (Resources)**: ゴールド, ルビー, ヒーローソウル(HS), エンシェントソウル(AS) などを自由に編集可能
- **ヒーロー管理 (Heroes)**: 全ヒーローのレベル変更、ギルド(Gild)設定、アップグレード全解放
- **古の神々 & 超越神 (Ancients & Outsiders)**: レベルを自在に調整
- **傭兵 & レリック (Mercenaries & Relics)**: 傭兵の復活・不死化、レリックのレアリティや効果編集
- **スタイリッシュなUI**: 従来のツールとは一線を画す、シンプルかつ洗練された「白と黒」のデザイン
- **安全設計**: チーター判定の解除機能、非同期通知(Toasts)による快適な操作

##  使い方 (Usage)
1. 上記リンク、または `index.html` をモダンブラウザで開く
2. **[ LOAD FILE ]** をクリックし、Clicker Heroesのセーブデータ (`.txt`) を読み込む
3. 各タブを切り替えて、ステータスやアイテムを編集
4. **[ SAVE DATA ]** をクリックして、編集済みデータをダウンロード
5. ゲーム本体にインポートして完了
